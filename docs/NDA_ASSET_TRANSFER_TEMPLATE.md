## 1. Confidentiality

Provider agrees to:
- Treat all Client data as strictly confidential
- Not disclose, copy, or transmit Client data to any third party
- Use Client data solely for the purpose of delivering the commissioned system
- Return or destroy all Client data within 30 days of contract termination

## 2. Data Access During Development

During the development period:
- Provider operates under time-limited access to Client infrastructure
- All access is logged and auditable by Client
- Provider personnel with access are listed in Appendix A
- Access is revoked immediately upon project completion or contract termination

## 3. Asset Transfer

Upon receipt of final payment, Provider transfers full ownership of:
- [ ] GCP project (including all Cloud Run services, secrets, IAM policies)
- [ ] Neon Postgres database (account ownership or data export)
- [ ] GitHub repository (or complete source code archive)
- [ ] All API keys, secrets, and credentials
- [ ] Documentation and operational runbooks
- [ ] Figma design files

Transfer timeline: within 5 business days of final payment confirmation.

## 4. Post-Transfer

After asset transfer:
- Provider retains ZERO access to Client data or infrastructure
- Provider retains rights to pre-existing IP (commissioning core patterns, not client-specific code)
- Client owns all deployment-specific code, data, and outputs
- Provider may reference the engagement in portfolio (without disclosing confidential details) unless Client objects in writing

## 5. AI Data Handling

During development:
- No Client financial data is sent to third-party AI providers in production
- Development/testing uses only synthetic or anonymized data with AI APIs
- Production classification uses self-hosted models on Client-owned infrastructure
- Provider will not train any AI model on Client data

## 6. Liability

Provider accepts liability for:
- Unauthorized disclosure of Client data
- Failure to transfer assets within the agreed timeline
- Security breaches caused by Provider negligence
